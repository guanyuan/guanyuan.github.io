使用jekyll

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

=> Now browse to http://localhost:4000 参考：http://jekyllrb.com/docs/quickstart/ 其中安装时，需要更换gem的仓库地址：

$ gem sources --remove http://rubygems.org/
$ gem sources -a http://ruby.taobao.org/
$ gem sources -l
*** CURRENT SOURCES ***
http://ruby.taobao.org   
$ gem install jekyll
